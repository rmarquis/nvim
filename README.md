# Neovim config

My Neovim config for Windows and Linux.

## Setup

Install:

* `neovim`

Clone the repository:

```shell
git clone git@github.com:rmarquis/nvim.git $HOME/.config/nvim
```

```powershell
git clone git@github.com:rmarquis/nvim.git $HOME/AppData/Local/nvim
```
